# Let Me Ask

A project where you as an admin, have the freedom to create rooms for your users to make questions to solve their problems. (is this stackoverflow?)

This was developed while following the NWL - Next Level Week, which is offered by [RocketSeat](https://rocketseat.com.br/).


## Technologies

This project was developed using the following:

* [Node](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/)
* [React](https://reactjs.org/)
* [Firebase](https://firebase.google.com/)
* [TypeScript](https://www.typescriptlang.org/)


## Getting started

clone, cd onto the folder and install the dependencies with yarn, then proceed to yarn start.
